## 产品名称

AI Translate

## 简介(≤132 字符,显示在搜索结果中)

在任意网页选中文字,立即获得 AI 翻译,还可选择查看易懂的学习讲解。

## 详细描述

**在任意网页上选中文字,立即获得 AI 翻译——如果你想真正学会这个词,还能获得更深入的讲解。**

**使用方法**
1. 在任意网页上选中文字。
2. 选中文字旁会出现一个小图标——点击它(或使用右键菜单中的"翻译所选文字")。
3. 立即查看翻译结果。想了解更多?点击"详解"查看释义、词性、CEFR 难度等级、由易到难的例句、常见搭配、近义词/反义词以及记忆技巧。

**开箱即用——无需设置,无需注册账号**
AI Translate 自带每日免费翻译额度,让你立即就能开始翻译,无需注册,无需 API 密钥。

**可自由选择翻译引擎**
- **免费试用**(默认):即时可用,无需设置。
- **设备端**(Chrome 138+):完全在你自己的电脑上私密翻译——文字绝不离开你的设备,也没有每日额度限制。
- **自备密钥**:在设置中填入你自己的 Gemini、OpenAI 或 DeepSeek API 密钥,即可无限量使用你偏好的服务商。

**专为语言学习者打造**
"详解"面板不只是查字典——它会生成从易到难分级的例句,让你在自己的水平上看到这个词的自然用法,再配上记忆技巧帮你牢记。

**隐私优先设计**
- 除非你主动选中文字并请求翻译,否则不会翻译任何内容——不会被动扫描你浏览的页面。
- 设备端引擎完全在本地处理文字。
- 如果你添加了自己的 API 密钥,它只会存储在你的浏览器本地,且仅用于直接与该服务商通信。
- 无统计分析,无广告,数据绝不出售给任何人。
- 完整隐私政策:请见本列表中的链接。

**支持 6 种语言**
本扩展的界面本身支持英语、简体中文、马来语、日语、越南语和韩语。
