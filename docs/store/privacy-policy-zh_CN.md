# 隐私政策 — AI Translate(翻译一下)

**最后更新:2026-07-03**

这是 AI Translate 浏览器扩展隐私政策的中文参考译本。英文版(`PRIVACY-POLICY.md`)为权威版本,如两者有出入,以英文版为准。

## 本扩展的功能

AI Translate 让你在任意网页上选中文字,即可获得 AI 翻译,并可选择查看"详解"面板(释义、例句、语法说明等)。这两项功能都由用户主动触发:除非你选中文字并点击翻译图标、右键菜单中的"翻译所选文字",或点击"详解"按钮,否则不会翻译、讲解或发送任何内容。

## 处理哪些数据、发往何处

本扩展仅处理以下数据:

- **你选中的文字**,以及所在段落的少量上下文(最多约 1200 个字符),用于让翻译/讲解结果更贴合语境。
- **你设置的目标语言**;使用"详解"功能时,还会在本地快速判断所选文字的源语言(基于文字/字符特征的简单启发式判断,完全在你的浏览器本地完成,不会外传)。

这些数据只会发送到**你当前使用的翻译引擎**,并且只在你主动请求翻译或详解的那一刻发送——绝不会自动发送、绝不会在后台发送,也绝不会针对你未选中的文字发送。

### 引擎一 — 免费试用(默认,无需设置)

- 你选中的文字及上下文会通过 HTTPS 发送到开发者自建的翻译网关(`gpt.yapweijun1996.com`),该网关会转发给一个 AI 语言模型以生成结果。
- 该网关对每次安装设有每日使用额度限制。它**仅记录使用次数**(用于额度控制),**不会**存储你请求的内容或返回的翻译/讲解结果——请求完成后,文字内容本身不会被保留。
- 达到每日额度后,扩展不会直接失败,而是提示你可以填入自己的 API 密钥(见下文),或第二天再试。

### 引擎二 — 设备端(隐私,无需联网)

- 使用 Chrome 内置的设备端翻译与语言检测 API。你的文字完全在你自己的设备上处理——**绝不会通过网络发送给任何人**,包括开发者本人。
- 这是唯一可以宣称"数据绝不离开设备"的引擎。
- 由于 Chrome 设备端 API 的技术限制,该引擎无法支持"详解"功能;当这是你唯一可用的引擎时,"详解"按钮会被禁用。

### 引擎三 — 自备密钥(Gemini、OpenAI 或 DeepSeek)

- 如果你在设置中填入自己的 Gemini、OpenAI 或 DeepSeek API 密钥,你选中的文字及上下文会**直接从你的浏览器发送到该服务商自己的 API**(分别为 `generativelanguage.googleapis.com`、`api.openai.com`、`api.deepseek.com`),使用你自己的密钥进行身份验证。
- 本扩展的开发者完全看不到这部分流量——数据直接从你的浏览器发往你选择的服务商。你的数据将适用该服务商自己的隐私政策与数据保留规则,而非本开发者的规则;如有需要,请自行查阅该服务商的政策。
- 一旦你为某个服务商填入密钥,免费试用网关将不再被使用。

## 存储了哪些数据、存于何处

| 数据 | 存储位置 | 是否会被发送 |
|---|---|---|
| 你填入的 API 密钥 | `chrome.storage.local`(仅你的浏览器本地) | 仅作为身份验证信息发送给对应服务商的 API——绝不发给开发者或任何其他人 |
| 你的引擎选择、目标语言 | `chrome.storage.local`(仅本地) | 否 |
| 近期翻译/详解结果的缓存(避免重复请求相同内容) | 浏览器本地 IndexedDB | 否——该缓存绝不离开你的设备 |

本扩展存储的任何数据都不会同步到开发者的服务器,也不会与第三方共享或出售。

## 本扩展不会做的事

- 不进行任何形式的统计分析或使用追踪。
- 没有广告,也绝不会将数据出售或分享给广告商。
- 不会被动读取页面内容——内容脚本只在你主动选中文字并点击按钮时才会运作。
- 使用免费试用或设备端引擎无需注册或登录账号。

## 为何需要"读取所有网站数据"的权限

Chrome 在安装时会提示本扩展可以"读取和更改你在所有网站上的数据"。这是因为扩展需要在**任意**你浏览的网页上检测文字选中动作,翻译图标才能随时出现——扩展无法预先知道你会在哪些网站上使用翻译功能。实际上,内容脚本只会读取你明确选中的文字,不会扫描、记录或发送页面上的其他任何内容。

## 儿童隐私

本扩展不面向 13 岁以下儿童,也不会有意收集其数据(如上所述,本扩展本身不收集任何人的个人数据)。

## 政策变更

如本政策发生影响数据处理方式或去向的变更,上方"最后更新"日期将随之更新;如有重大变更,还会在 Chrome 应用商店的扩展页面中另行说明。
