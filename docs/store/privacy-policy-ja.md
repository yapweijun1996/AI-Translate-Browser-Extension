# プライバシーポリシー — AI Translate

**最終更新日:2026-07-03**

これは AI Translate ブラウザ拡張機能のプライバシーポリシーの日本語参考訳です。英語版(`PRIVACY-POLICY.md`)が正式版であり、内容に相違がある場合は英語版が優先されます。

## この拡張機能について

AI Translate は、任意のウェブページでテキストを選択すると AI による翻訳が得られ、さらに任意で選択した語句の「詳しく解説」パネル(定義、例文、文法の説明など)を表示できる拡張機能です。どちらの機能もユーザーの操作によってのみ起動します。テキストを選択して翻訳アイコン、右クリックメニューの「選択したテキストを翻訳」、または「詳しく解説」ボタンをクリックしない限り、何も翻訳・解説・送信されることはありません。

## 処理されるデータとその送信先

この拡張機能が処理するのは以下のデータのみです。

- **選択したテキスト**と、その段落の前後の短い文脈(最大約1,200文字)。翻訳・解説の精度を文脈に合わせて高めるために使用します。
- **翻訳先言語の設定**。「詳しく解説」機能使用時には、選択テキストの元言語をローカルで簡易的に推定します(使用されている文字種に基づく簡単な判定で、ブラウザ外へは一切送信されません)。

これらのデータは**現在使用しているエンジン1つだけ**に送信され、それも翻訳・解説をリクエストした瞬間のみです。自動送信やバックグラウンドでの送信、選択していないテキストの送信は一切ありません。

### エンジン1 — 無料お試し(デフォルト、設定不要)

- 選択したテキストと文脈は、開発者が運営する翻訳ゲートウェイ(`gpt.yapweijun1996.com`)へ HTTPS 経由で送信され、そこから AI 言語モデルへ中継されて結果が生成されます。
- このゲートウェイはインストールごとに1日あたりの利用上限を設けています。記録されるのは**利用回数のみ**(上限管理のため)で、リクエストの内容や翻訳・解説結果そのものは保存されません。リクエストへの応答が完了すると、テキスト自体は保持されません。
- 1日の上限に達しても、拡張機能は単にエラーを表示するのではなく、自分の API キーを設定する(下記参照)か翌日また試すかを選べるようにします。

### エンジン2 — オンデバイス(プライベート、ネットワーク不使用)

- Chrome 内蔵のオンデバイス翻訳・言語検出 API を使用します。テキストはすべてお使いの端末内で処理され、**開発者を含め、誰にもネットワーク経由で送信されることはありません**。
- 「データが端末外に一切出ない」と言えるのはこのエンジンのみです。
- Chrome のオンデバイス API の技術的制約により、このエンジンは「詳しく解説」機能を利用できません。これが唯一利用可能なエンジンの場合、「詳しく解説」ボタンは無効化されます。

### エンジン3 — 自分のAPIキーを使用(Gemini、OpenAI、DeepSeek)

- 設定画面で Gemini、OpenAI、DeepSeek いずれかの自分の API キーを追加した場合、選択したテキストと文脈は**あなたのブラウザからそのプロバイダー自身の API へ直接**送信されます(それぞれ `generativelanguage.googleapis.com`、`api.openai.com`、`api.deepseek.com`)。認証にはあなた自身のキーが使用されます。
- この拡張機能の開発者はこの通信を一切閲覧できません — データはあなたのブラウザから選択したプロバイダーへ直接送られます。データの扱いは開発者の方針ではなく、そのプロバイダー自身のプライバシーポリシーおよびデータ保持方針に従います。気になる場合は各プロバイダーのポリシーを直接ご確認ください。
- いずれかのプロバイダーのキーを追加すると、無料お試しゲートウェイは以後一切使用されません。

## 保存されるデータとその保存場所

| データ | 保存場所 | どこかへ送信されるか |
|---|---|---|
| 追加した API キー | `chrome.storage.local`(お使いのブラウザ内のみ) | 該当プロバイダーの API への認証情報としてのみ送信 — 開発者やその他へは一切送信されません |
| エンジンの選択、翻訳先言語 | `chrome.storage.local`(ローカルのみ) | いいえ |
| 最近の翻訳・解説結果のキャッシュ(同じテキストの再リクエストを避けるため) | ブラウザ内 IndexedDB | いいえ — このキャッシュは端末外へ一切出ません |

この拡張機能が保存するデータは、開発者が運営するサーバーへ同期されることはなく、第三者と共有・販売されることもありません。

## この拡張機能が行わないこと

- いかなる分析・テレメトリ・利用状況の追跡も行いません。
- 広告は一切なく、データが広告主に販売・共有されることもありません。
- ページ内容を受動的に読み取ることはありません — コンテンツスクリプトは、あなたが行った選択とクリックした操作にのみ反応します。
- 無料お試し・オンデバイスエンジンの利用にアカウント登録やログインは不要です。

## 「すべてのウェブサイトのデータを読み取る」権限が必要な理由

Chrome はインストール時に、この拡張機能が「すべてのウェブサイトであなたのデータを読み取り、変更できる」という警告を表示します。これは、**どのページ**を閲覧していてもテキスト選択を検知し、翻訳アイコンを表示できるようにする必要があるためです — どのサイトで翻訳機能を使うかを事前に知ることはできません。実際には、コンテンツスクリプトはあなたが明示的に選択したテキストのみを読み取り、ページ内のそれ以外の内容をスキャン・記録・送信することはありません。

## 児童のプライバシー

この拡張機能は13歳未満の児童を対象としておらず、児童のデータを意図的に収集することもありません(上記の通り、年齢を問わず個人データの収集自体を行っていません)。
