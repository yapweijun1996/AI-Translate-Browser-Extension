# Chrome 웹 스토어 등록정보 — 한국어

## 제품 이름

AI Translate

## 요약(132자 이내, 검색 결과에 표시)

어떤 페이지에서든 텍스트를 선택하면 즉시 AI 번역, 원하면 학습자 친화적인 설명도 함께.

## 상세 설명

**어떤 웹페이지에서든 텍스트를 선택하면 즉시 AI 번역을 받을 수 있습니다 — 단어를 제대로 배우고 싶다면 더 깊이 있는 설명도 볼 수 있습니다.**

**사용 방법**
1. 어떤 페이지에서든 텍스트를 선택하세요.
2. 선택 영역 옆에 작은 아이콘이 나타납니다 — 클릭하세요(또는 우클릭 메뉴의 "선택한 텍스트 번역" 사용).
3. 즉시 번역 결과를 확인하세요. 더 알고 싶다면 "자세히 설명"을 클릭해 정의, 품사, CEFR 난이도, 쉬운 것부터 어려운 것까지 단계별 예문, 자주 쓰이는 조합, 유의어/반의어, 암기 팁까지 확인할 수 있습니다.

**바로 사용 가능 — 설정도 계정도 필요 없음**
AI Translate에는 매일 무료 번역 한도가 기본 제공되어 즉시 번역을 시작할 수 있습니다. 가입도, API 키도 필요 없습니다.

**원하는 엔진을 선택하세요**
- **무료 체험판**(기본값): 설정 없이 즉시 사용 가능.
- **온디바이스**(Chrome 138 이상): 자신의 컴퓨터에서만 완전히 비공개로 번역 — 텍스트가 기기를 벗어나지 않으며, 일일 한도도 없습니다.
- **직접 API 키 사용**: 설정에서 자신의 Gemini, OpenAI, DeepSeek API 키를 연결하면 원하는 제공업체로 무제한 사용할 수 있습니다.

**언어 학습자를 위해 설계됨**
"자세히 설명" 패널은 단순한 사전 검색이 아닙니다 — 쉬운 것부터 어려운 것까지 단계별 예문을 생성해, 자신의 수준에서 단어가 자연스럽게 쓰이는 모습을 확인할 수 있고, 암기 팁으로 기억에도 도움을 줍니다.

**개인정보 보호를 고려한 설계**
- 텍스트를 선택하고 번역을 요청하기 전까지는 아무것도 번역되지 않습니다 — 방문하는 페이지를 수동적으로 스캔하지 않습니다.
- 온디바이스 엔진은 텍스트를 완전히 로컬에서 처리합니다.
- 직접 API 키를 추가한 경우, 이는 사용자의 브라우저에만 저장되며 해당 제공업체와 직접 통신하는 용도로만 사용됩니다.
- 분석 없음, 광고 없음, 누구에게도 데이터를 판매하지 않습니다.
- 전체 개인정보처리방침: 이 등록정보의 링크를 참고하세요.

**6개 언어 지원**
이 확장 프로그램 자체의 인터페이스는 영어, 중국어 간체, 말레이어, 일본어, 베트남어, 한국어를 지원합니다.
