# Penyenaraian Chrome Web Store — Bahasa Melayu

## Nama produk

AI Translate

## Ringkasan (≤132 aksara, dipaparkan dalam hasil carian)

Pilih teks pada mana-mana halaman untuk terjemahan AI segera, ditambah penjelasan mesra pelajar pilihan.

## Penerangan terperinci

**Pilih mana-mana teks pada mana-mana laman web. Dapatkan terjemahan AI segera — dan jika anda mahu benar-benar mempelajari perkataan itu, penjelasan yang lebih mendalam juga.**

**Cara ia berfungsi**
1. Pilih teks pada mana-mana halaman.
2. Ikon kecil muncul di sebelah pilihan anda — klik ia (atau guna menu klik-kanan "Terjemah teks yang dipilih").
3. Lihat terjemahan serta-merta. Mahu lebih? Klik "Terangkan" untuk definisi, jenis kata, tahap CEFR, contoh ayat bergred, kolokasi biasa, sinonim/antonim, dan petua mengingat.

**Berfungsi terus — tiada persediaan, tiada akaun**
AI Translate disertakan dengan kuota terjemahan percuma harian, jadi anda boleh mula menterjemah serta-merta. Tiada pendaftaran, tiada kunci API diperlukan.

**Pilihan enjin anda sendiri**
- **Percubaan percuma** (lalai): berfungsi serta-merta, tiada persediaan.
- **Dalam peranti** (Chrome 138+): terjemah secara persendirian, sepenuhnya pada komputer anda sendiri — teks tidak pernah meninggalkan peranti anda, tiada had harian.
- **Guna kunci anda sendiri**: sambungkan kunci API Gemini, OpenAI, atau DeepSeek anda sendiri dalam Tetapan untuk penggunaan tanpa had dengan pembekal pilihan anda.

**Dibina untuk pelajar bahasa**
Panel "Terangkan" bukan sekadar carian kamus — ia menjana contoh ayat bergred dari mudah ke sukar, supaya anda dapat melihat perkataan itu digunakan secara semula jadi pada tahap anda, ditambah petua mengingat untuk membantu ia melekat di ingatan.

**Direka dengan privasi sebagai keutamaan**
- Tiada apa diterjemah sehingga anda memilih teks dan memilih untuk menterjemahnya — tiada imbasan pasif laman yang anda lawati.
- Enjin dalam peranti memproses teks sepenuhnya secara tempatan.
- Kunci API anda sendiri, jika ditambah, disimpan hanya dalam pelayar anda dan digunakan hanya untuk berkomunikasi terus dengan pembekal itu.
- Tiada analitik, tiada iklan, tiada data dijual kepada sesiapa.
- Dasar privasi penuh: lihat pautan dalam penyenaraian ini.

**Tersedia dalam 6 bahasa**
Antara muka sambungan ini sendiri tersedia dalam Bahasa Inggeris, Cina Ringkas, Bahasa Melayu, Jepun, Vietnam, dan Korea.
