# 개인정보처리방침 — AI Translate

**최종 업데이트: 2026-07-03**

이 문서는 AI Translate 브라우저 확장 프로그램 개인정보처리방침의 한국어 참고 번역본입니다. 영어 원본(`PRIVACY-POLICY.md`)이 공식 버전이며, 내용이 상충할 경우 영어 원본이 우선합니다.

## 이 확장 프로그램이 하는 일

AI Translate는 어떤 웹페이지에서든 텍스트를 선택하면 AI 번역을 받을 수 있고, 선택한 구문에 대한 "자세히 설명" 패널(정의, 예문, 문법 설명 등)을 선택적으로 볼 수 있게 해줍니다. 두 기능 모두 사용자가 직접 실행해야만 작동합니다 — 텍스트를 선택하고 번역 아이콘, 우클릭 메뉴의 "선택한 텍스트 번역", 또는 "자세히 설명" 버튼을 클릭하기 전까지는 아무것도 번역·설명·전송되지 않습니다.

## 처리되는 데이터와 전송 위치

이 확장 프로그램이 처리하는 데이터는 다음뿐입니다.

- **선택한 텍스트**와 해당 문단의 짧은 주변 문맥(최대 약 1,200자) — 번역·설명을 문맥에 맞게 더 정확하게 만들기 위해 사용됩니다.
- **설정한 번역 대상 언어**, 그리고 "자세히 설명" 기능 사용 시 선택한 텍스트의 원본 언어에 대한 간단한 로컬 추정(사용된 문자 특성에 기반한 간단한 판단으로, 브라우저 밖으로 전송되지 않습니다).

이 데이터는 **현재 사용 중인 번역 엔진 단 하나**에만 전송되며, 번역이나 설명을 요청하는 그 순간에만 전송됩니다 — 자동으로도, 백그라운드에서도, 선택하지 않은 텍스트에 대해서도 절대 전송되지 않습니다.

### 엔진 1 — 무료 체험판(기본값, 설정 불필요)

- 선택한 텍스트와 문맥은 HTTPS를 통해 개발자가 운영하는 번역 게이트웨이(`gpt.yapweijun1996.com`)로 전송되며, 이 게이트웨이가 AI 언어 모델로 중계하여 결과를 생성합니다.
- 이 게이트웨이는 설치본별로 일일 사용 한도를 적용합니다. **사용 횟수만** 기록하며(한도 관리 목적), 요청 내용이나 반환된 번역·설명 결과는 저장하지 않습니다. 요청에 대한 응답이 완료되면 텍스트 자체는 보관되지 않습니다.
- 일일 한도에 도달해도 확장 프로그램은 단순히 오류를 표시하지 않고, 자신의 API 키를 설정하거나(아래 참고) 다음 날 다시 시도할 수 있는 선택지를 제공합니다.

### 엔진 2 — 온디바이스(비공개, 네트워크 미사용)

- Chrome에 내장된 온디바이스 번역 및 언어 감지 API를 사용합니다. 텍스트는 전적으로 사용자의 기기에서 처리되며, **개발자를 포함해 누구에게도 네트워크를 통해 전송되지 않습니다**.
- "데이터가 기기를 절대 벗어나지 않는다"고 말할 수 있는 유일한 엔진입니다.
- Chrome 온디바이스 API의 기술적 제약으로 인해 이 엔진은 "자세히 설명" 기능을 지원할 수 없습니다. 이것이 유일하게 사용 가능한 엔진일 경우 "자세히 설명" 버튼은 비활성화됩니다.

### 엔진 3 — 직접 API 키 사용(Gemini, OpenAI, DeepSeek)

- 설정에서 Gemini, OpenAI, DeepSeek 중 자신의 API 키를 추가하면, 선택한 텍스트와 문맥은 **사용자의 브라우저에서 해당 제공업체 자체 API로 직접** 전송됩니다(각각 `generativelanguage.googleapis.com`, `api.openai.com`, `api.deepseek.com`). 인증에는 사용자 본인의 키가 사용됩니다.
- 이 확장 프로그램의 개발자는 이 트래픽을 전혀 볼 수 없습니다 — 데이터는 사용자의 브라우저에서 선택한 제공업체로 곧바로 전송됩니다. 이 데이터의 처리는 개발자가 아닌 해당 제공업체 자체의 개인정보처리방침과 데이터 보관 정책을 따릅니다. 필요하다면 해당 제공업체의 정책을 직접 확인하시기 바랍니다.
- 특정 제공업체의 키를 추가하면 무료 체험판 게이트웨이는 더 이상 전혀 사용되지 않습니다.

## 저장되는 데이터와 저장 위치

| 데이터 | 저장 위치 | 어딘가로 전송되는가 |
|---|---|---|
| 추가한 API 키 | `chrome.storage.local`(사용자의 브라우저에만) | 해당 제공업체 API에 대한 인증 헤더로만 전송 — 개발자나 그 외 어디에도 전송되지 않음 |
| 엔진 선택, 번역 대상 언어 | `chrome.storage.local`(로컬에만) | 아니요 |
| 최근 번역·설명 결과 캐시(동일 텍스트 재요청 방지) | 브라우저 내 IndexedDB | 아니요 — 이 캐시는 기기를 절대 벗어나지 않습니다 |

이 확장 프로그램이 저장하는 어떤 데이터도 개발자가 운영하는 서버로 동기화되지 않으며, 제3자와 공유되거나 판매되지 않습니다.

## 이 확장 프로그램이 하지 않는 것

- 어떠한 형태의 분석, 텔레메트리, 사용 추적도 하지 않습니다.
- 광고가 없으며, 데이터가 광고주에게 판매되거나 공유되는 일도 없습니다.
- 페이지 콘텐츠를 수동적으로 읽지 않습니다 — 콘텐츠 스크립트는 사용자가 직접 선택하고 클릭한 동작에만 반응합니다.
- 무료 체험판이나 온디바이스 엔진 사용에 계정 등록이나 로그인이 필요하지 않습니다.

## "모든 웹사이트의 데이터 읽기" 권한이 필요한 이유

Chrome은 설치 시 이 확장 프로그램이 "모든 웹사이트에서 사용자의 데이터를 읽고 변경할 수 있다"는 경고를 표시합니다. 이는 사용자가 읽는 **어떤** 페이지에서든 텍스트 선택을 감지해서 번역 아이콘이 나타날 수 있도록 해야 하기 때문입니다 — 어떤 사이트에서 번역 기능을 사용할지 미리 알 수 없습니다. 실제로 콘텐츠 스크립트는 사용자가 명시적으로 선택한 텍스트만 읽으며, 페이지의 다른 내용을 스캔·기록·전송하지 않습니다.
