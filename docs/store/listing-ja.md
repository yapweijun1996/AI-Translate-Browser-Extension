# Chrome ウェブストア掲載情報 — 日本語

## 製品名

AI Translate

## 概要(132文字以内、検索結果に表示)

任意のページでテキストを選択するだけで即座にAI翻訳。学習者向けのわかりやすい解説も選べます。

## 詳細説明

**どのウェブページでもテキストを選択するだけで、即座にAI翻訳が得られます。単語をしっかり学びたいときは、さらに詳しい解説も。**

**使い方**
1. 任意のページでテキストを選択します。
2. 選択箇所の近くに小さなアイコンが表示されるので、クリックします(または右クリックメニューの「選択したテキストを翻訳」を使用)。
3. すぐに翻訳結果が表示されます。もっと知りたい場合は「詳しく解説」をクリックすると、定義・品詞・CEFRレベル・易しいものから難しいものへ段階的な例文・よく使われる組み合わせ・類義語/対義語・覚え方のヒントが表示されます。

**すぐに使える — 設定もアカウントも不要**
AI Translateには無料の1日利用枠が付属しているので、すぐに翻訳を始められます。登録もAPIキーも不要です。

**翻訳エンジンを自由に選択**
- **無料お試し**(デフォルト):設定不要ですぐに使えます。
- **オンデバイス**(Chrome 138以降):お使いのパソコン内で完全にプライベートに翻訳——テキストが端末外に出ることはなく、1日の上限もありません。
- **自分のAPIキーを使用**:設定画面で自分のGemini、OpenAI、DeepSeekのAPIキーを接続すれば、お好みのプロバイダーで無制限に利用できます。

**語学学習者のために設計**
「詳しく解説」パネルは単なる辞書検索ではありません。易しいものから難しいものへと段階的な例文を生成するので、自分のレベルでその単語の自然な使い方を確認でき、覚え方のヒントで記憶にも残ります。

**プライバシーに配慮した設計**
- テキストを選択して翻訳を選ぶまでは何も翻訳されません——閲覧しているページを受動的にスキャンすることはありません。
- オンデバイスエンジンはテキストを完全にローカルで処理します。
- 自分のAPIキーを追加した場合、それはお使いのブラウザ内にのみ保存され、そのプロバイダーと直接やり取りする用途にのみ使用されます。
- アクセス解析なし、広告なし、データを誰かに販売することもありません。
- プライバシーポリシー全文:この掲載情報内のリンクをご覧ください。

**6言語対応**
この拡張機能自体のインターフェースは、英語、簡体字中国語、マレー語、日本語、ベトナム語、韓国語に対応しています。
