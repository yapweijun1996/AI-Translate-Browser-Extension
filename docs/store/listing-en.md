## Detailed description

**Select any text on any webpage. Get an instant AI translation — and, if you want to actually learn the word, a deeper explanation too.**

**How it works**
1. Select text on any page.
2. A small icon appears next to your selection — click it (or use the right-click "Translate selection" menu).
3. See the translation instantly. Want more? Click "Explain" for the definition, part of speech, CEFR level, graded example sentences, common collocations, synonyms/antonyms, and a memory tip.

**Works out of the box — no setup, no account**
AI Translate ships with a free daily translation allowance, so you can start translating immediately. No sign-up, no API key required.

**Your choice of engine**
- **Free trial** (default): works instantly, no setup.
- **On-device** (Chrome 138+): translate privately, entirely on your own computer — no text ever leaves your device, no daily limit.
- **Bring your own key**: connect your own Gemini, OpenAI, or DeepSeek API key in Settings for unlimited use with your preferred provider.

**Built for language learners**
The "Explain" panel isn't just a dictionary lookup — it generates example sentences graded from easy to hard, so you can see the word used naturally at your level, plus a memory tip to help it stick.

**Privacy by design**
- Nothing is translated until you select text and choose to translate it — no passive scanning of pages you visit.
- The on-device engine processes text entirely locally.
- Your own API keys, if you add one, are stored only in your browser and used only to talk directly to that provider.
- No analytics, no ads, no data sold to anyone.
- Full privacy policy: see the link in this listing.

**Available in 6 languages**
The extension's own interface is available in English, Simplified Chinese, Malay, Japanese, Vietnamese, and Korean.
